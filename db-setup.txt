1. Pull the docker image for MySQL
    >docker pull mysql/mysql-server:latest
2. Run the docker image and map the external port 3306 to the db port 3306
    >docker run --name=mysql1 -e MYSQL_ROOT_HOST=% -p 3306:3306 -d mysql/mysql-server
3. Read the logs from the Docker container to find the initial, autogenerated password for the MySQL root applicationUser
    >docker logs mysql1 2>&1
4. Change the autogenerated password
    >docker exec -it mysql1 mysql -u root -p
    >[Enter password]
    >alter user 'root'@'localhost' identified by '<new-password>';
5. Create the database for the project
    >create database db_diplomska;
6. Create admin applicationUser for the database created in the previous step and grant all permissions
    >create user 'diplomska-admin'@'%' identified by '<password>';
    >grant all on db_diplomska.* to 'diplomska-admin'@'%';

*. Restart the container if it is stopped:
    >sudo docker start `sudo docker ps -a -q --filter "name=mysql1"`
